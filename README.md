# Shiperist UI components library

## Description 

A set of many UI components based on Catpuccin color theme, made in React, currently contains 22 components, some of them have sub-components

## Features

- Responsive
- Clean code
- Exists as package on npm

- Badge

- Kbd
- Layout
- Menu

- Text-input
- Textarea

- Typography
